- import json (form: [{"word_key": "word", "title_vals": {"title0": val0, ..., "titleN": valN}}, ..., {...}])
- need to install extra mongo tools (download/install process probably different in ubuntu)
- locally I'm doing all mongo stuff in windows because that's how I've installed everything
- installation docs: https://docs.mongodb.com/database-tools/installation/installation-linux/

- mongo files (local) located:
C:/"Program Files"/MongoDB/server/4.4/bin

- run mongo cmd
C:/"Program Files"/MongoDB/server/4.4/bin/mongo.exe

- this should create a db of dbName with a collection of collectionName from the json file
mongoimport --db dbName --collection collectionName --file fileName.json --jsonArray

- example imports (local):
C:/"Program Files"/MongoDB/Server/4.4/bin/mongoimport.exe --db bookData --collection titleMetadata --file C:/Users/kyler/pyguy/bookOracle/v6/data/title_data_json_arrays/02_13_2021_set0_0_100.json --jsonArray
C:/"Program Files"/MongoDB/Server/4.4/bin/mongoimport.exe --db bookData --collection wordVals --file C:/Users/kyler/pyguy/bookOracle/v6/data/master_dict_json_arrays/02_13_2021_set0_0_100.json --jsonArray

- switch to db (--db given above was bookData)
use bookData

- view collections in db
show collections

- query db for word "call" (--collection given above was wordVals)
db.wordVals.find({word_key: "call"})

- drop collection, collectionName is the name of the collection
db.collectionName.drop()
